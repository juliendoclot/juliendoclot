Ce thème est livré avec l'intégration [@astrojs/mdx](https://docs.astro.build/en/guides/integrations-guide/mdx/) installée et configurée dans votre fichier `astro.config.mjs`. Si vous préférez ne pas utiliser MDX, vous pouvez désactiver le support en supprimant l'intégration de votre fichier de configuration.

## Pourquoi MDX ?

MDX est une version spéciale de Markdown qui supporte la syntaxe JavaScript et JSX embarquée. Cela permet de [mélanger JavaScript et composants UI dans votre contenu Markdown](https://docs.astro.build/en/guides/integrations-guide/mdx/#mdx-in-astro) pour créer des graphiques interactifs ou des alertes.

Si vous avez du contenu existant écrit en MDX, cette intégration devrait faciliter la migration vers Astro.

## Exemple

Voici comment importer et utiliser un composant UI dans MDX.
Lorsque vous ouvrez cette page dans le navigateur, vous devriez voir le bouton cliquable ci-dessous.

import HeaderLink from '../../components/HeaderLink.astro';

<div class="my-4">
<HeaderLink href="#" onclick="alert('Cliqué !')">
	Composant embarqué dans MDX
</HeaderLink>
</div>
